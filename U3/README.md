**秒**  
|單位名稱|全名|實際大小|
|-------|---|--------|
|毫秒（ms）|millisecond| 1ms = 10-3秒|
|微秒（μs)|microsecond|1μs = 10-6秒|
|奈秒（ns）|nanosecond |1ns = 10-9秒|
|披秒（ps）|picosecond| 1ps= 10-12秒|
